### 👍 3. 新增点赞类型选择功能

在"朋友圈"页面的"点赞设置"区域新增 3 个复选框：

| 选项         | 说明                   | 默认状态 |
| ------------ | ---------------------- | -------- |
| ✿ 文字朋友圈 | 点赞纯文字内容的朋友圈 | ✓ 勾选   |
| ♡ 图片朋友圈 | 点赞包含图片的朋友圈   | ✓ 勾选   |
| ★ 视频朋友圈 | 点赞包含视频的朋友圈   | ✓ 勾选   |

#### 功能特点：

- 用户可以自由组合选择，例如：
  - 只点赞图片：勾选 ♡ 图片朋友圈 ，取消其他
  - 只点赞视频：勾选 ★ 视频朋友圈，取消其他
  - 点赞文字和图片：勾选前两个，取消视频
- **至少要选择一种类型**，否则点赞会提示错误
- 选择状态会自动保存到配置文件

## 🎯 使用建议

### 基础使用流程

1. 打开程序，进入"💖 朋友圈"页面
2. 选择点赞模式：
   - "只给特定人点赞" - 输入要点赞的用户名
   - "给所有人点赞" - 可选设置黑名单
3. **勾选要点赞的内容类型**（文字/图片/视频）
4. 设置点赞间隔（默认 5 秒）
5. 可选：启用评论功能
6. 点击"▶ 开始朋友圈功能"开始执行

### 场景示例

- **只给朋友的图片点赞**：选择特定人模式 + 只勾选 ♡ 图片
- **给所有人的视频点赞**：选择所有人模式 + 只勾选 ★ 视频
- **点赞所有内容**：勾选全部三种类型
